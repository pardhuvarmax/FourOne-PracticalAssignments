# Task 1: LLM Workflow (5 Marks)

Develop a Python program that accepts user input and generates a response using an LLM (OpenAI/Gemini/Ollama).

## How it Works
1. Imports the shared LLM wrapper (`utils.llm`) configured with your Gemini API key.
2. Prompts the user for queries via an interactive command-line interface (CLI) loop.
3. Requests and displays the LLM response in real-time.

## Running Instructions
Run the script using the project's virtual environment:
```bash
./venv/bin/python task_1_llm_workflow/main.py
```
To exit the interactive session, type `exit`, `quit`, or `q`.
